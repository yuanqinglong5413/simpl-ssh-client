- **技术栈**: 采用 React + TypeScript 构建 UI，通过 `@tauri-apps/api` 与 Rust 后端进行 IPC 通信（`invoke`/`listen`）。
- **核心架构**: `App.tsx` 作为根组件，集中管理会话（Sessions）、标签页（Tabs）及连接配置（Profiles）的状态。
- **布局系统**: 实现基于 `SplitNode` 递归树的自定义分屏系统（`SplitView`），支持终端面板的动态水平/垂直分割与拖拽调整。
- **主题与设置**: 通过 `ThemeProvider` 和 `SettingsProvider` 利用 Context API 全局管理应用主题（CSS 变量注入）及用户偏好，并持久化至 `localStorage`。
- **功能模块**: 模块化封装了 `TerminalPane`（集成 xterm.js 与 WebSocket）、`SftpPane`、`MonitorPane`、`EditorPane` 及 `GitPanel` 等业务组件。
- **状态恢复**: 利用 `useWorkspaceRestore` Hook 在启动时从后端加载工作区快照并自动重连，同时在标签页变化时防抖保存状态。
