- **SFTP 会话缓存**：`sftp.rs` 定义 `SftpManager`，在已建立的 `russh` 会话上打开 SFTP subsystem channel 并缓存 `SftpSession`，避免重复认证。
- **异步传输队列**：`transfer.rs` 实现 `TransferQueue`，采用串行 worker 模式处理上传/下载任务，支持原子取消标志（`AtomicBool`）和实时进度事件推送（`transfer://progress`）。
- **目录同步引擎**：`sync.rs` 提供 `run_directory_sync`，通过并行扫描本地与远程目录树生成差异计划，并按时间戳比对结果将任务自动注入传输队列。
- **Git 命令代理**：`git_ops.rs` 封装 `exec_git` helper，复用 SSH exec channel 执行 git 命令，并提供针对 `porcelain` 格式输出的结构化解析器（status, log, diff 等）。
