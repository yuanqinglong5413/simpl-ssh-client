- **启动编排**：在 `src/lib.rs` 的 `tauri::Builder` 中集中注册所有子模块的状态管理器（如 `SessionManager`, `SftpManager`），并通过 `.setup()` 钩子初始化本地 WebSocket 桥接服务与异步传输队列。
- **通信契约**：采用 Tauri IPC 机制，由 `commands.rs` 定义统一的命令接口层，将前端请求分发至 `session/` 下的各功能模块。
- **资源共享**：通过 `SessionManager` 维护持久化的 `russh::client::Handle`，实现终端 PTY、SFTP 通道、端口转发及 Git 操作对底层 SSH 连接的零拷贝复用。
- **安全隔离**：利用 `HostKeyVerifier` 与 `ProfileStore` 结合系统钥匙串（`keyring`）实现敏感凭证的加密存储与主机公钥的动态校验流程。
